### Question

**7. Standing Wave Modes**
A standing wave with four antinodes is produced on a string of length L = 80 cm. What is the wavelength of this wave?

---

### Context: What is a Standing Wave?

Imagine plucking a guitar string or vibrating a jump rope tied to a wall. The wave travels down the line, hits the fixed end, and reflects back. When the original wave and the reflected wave perfectly overlap and interfere with one another, they create a **standing wave**. Instead of looking like it's traveling forward, the wave appears to just vibrate up and down in place.

* The "humps" of maximum movement (where the string moves up and down the most) are called **antinodes**.
* The completely still "pinch points" between the humps (where the string doesn't move at all) are called **nodes**.

---

### Notes & Key Concepts

To solve this problem, we rely on the geometry of these waves on a string fixed at both ends:

* **The Harmonic Number (n):** The number of antinodes (humps) is exactly equal to the harmonic number. In this problem, 4 antinodes mean it is the 4th harmonic (n = 4).

* **The Length of a Loop:** Each individual "loop" (the section from one node to the next, containing exactly one antinode) makes up exactly half of a full wavelength (λ / 2).

* **The Formula:** The total length of the string (L) is equal to the number of loops (n) multiplied by the physical length of one loop.
    * `L = n * (λ / 2)`

---

### Step-by-Step Solution

**Step 1: Identify the given information.**
* Total length of the string (L) = 80 cm
* Number of antinodes (n) = 4

**Step 2: Set up the standing wave equation.**
* `L = n * (λ / 2)`

**Step 3: Substitute the known values into the equation.**
* `80 = 4 * (λ / 2)`

**Step 4: Simplify and solve for the wavelength (λ).**
* First, simplify the right side of the equation (4 divided by 2 is 2):
    `80 = 2 * λ`

* Next, divide both sides by 2 to isolate λ:
    `λ = 80 / 2`
    `λ = 40`

---

### Final Result

**Wavelength (λ) = 40 cm**
